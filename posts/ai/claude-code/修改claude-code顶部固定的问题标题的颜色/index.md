# 修改Claude Code顶部固定的问题标题的颜色

需要自定义主题
1. /theme并创建定义主题
2. 搜索subtle
3. 修改为RGB(253, 186, 118)

或者创建自定义主题后直接修改~\.claude\themes\theme.json
